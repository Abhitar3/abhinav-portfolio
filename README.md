# Abhinav Reddy Terupally Portfolio

A clean, modern, responsive portfolio website built with React, Vite, Tailwind CSS, and JavaScript.

## Install dependencies

```bash
npm install
```

## Run locally

```bash
npm run dev
```

## Build

```bash
npm run build
```

## Preview production build

```bash
npm run preview
```

## Deployment

### Vercel

Import this repository into Vercel. Use the default Vite settings:

- Build command: `npm run build`
- Output directory: `dist`

### Netlify

Import this repository into Netlify and use:

- Build command: `npm run build`
- Publish directory: `dist`

### GitHub Pages

Build the project with `npm run build`, then deploy the generated `dist` folder using your preferred GitHub Pages workflow.

## Portfolio content

Editable content lives in:

- `src/data/projects.js`
- `src/data/skills.js`
- `src/data/experience.js`

## Resume and report files

Place these files in the `public` folder:

- `public/Developer-Experience-AI.pdf`
- `public/Abhinav_Reddy_Resume.pdf`

They will be served at:

- `/Developer-Experience-AI.pdf`
- `/Abhinav_Reddy_Resume.pdf`
